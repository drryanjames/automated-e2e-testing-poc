# automated-e2e-testing-poc

## Getting Started
(instructions take from [here](https://www.protractortest.org/#/))

Install protractor
```
npm install -g protractor
```

Update webdriver
```
webdriver-manager update
```

Start the server
```
webdriver-manager start
```

Run the tests
```
cd automated-e2e-testing-poc
npm test
```
